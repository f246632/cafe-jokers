## 🚀 Quick Start

### View Locally

```bash
# Clone the repository
git clone https://github.com/f246632/cafe-jokers.git

# Navigate to directory
cd cafe-jokers

# Start a local server (Python 3)
python3 -m http.server 8000

# Visit http://localhost:8000
```
